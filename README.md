# Sistema de jogo de xadrez

O projeto foi desenvolvido utilizando programação orientada a objetos com C#.

### Técnicas aplicadas:
- Construtores
- Palavra this
- Sobrecarga
- Encapsulamento
- Associação
- Herança
- Classe abstrata
- Sobreposição
- Polimorfismo
- Elementos estáticos
- Tipos enumerados
- Exceções

### Estruturas de dados aplicadas:
- Vetores
- Matrizes
- Listas
- Conjuntos
